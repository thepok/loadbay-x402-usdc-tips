How a tip works

  1. Click the tip button. The widget fetches /x402/health from the live x402 provider.
  2. The provider returns an HTTP 402 Payment Required challenge including the exact payTo, amount, network, and asset.
  3. An x402-aware wallet (Coinbase, PayAI, Bankr, thirdweb, or a self-custody browser wallet) settles 0.001 USDC on Base directly to the configured address.
  4. The provider returns the protected payload (or a 200 on health). No human operator step, no OAuth, no KYC.
